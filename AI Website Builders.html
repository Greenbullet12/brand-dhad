Here are the exact steps to publish your free website for brand using GitHub Pages
No payment, no hosting fees, no coding needed beyond copy-paste.

BEFORE YOU START

You need:

STEP 1 Create a GitHub Account

  1. Go to github.com

  2. Click Sign up

  3. Choose a username (e.g. "brandDhad")

  4. Enter email and password

  5. Complete the CAPTCHA and email verification

  6. Skip all setup questions

  7. Youre now on your dashboard

STEP 2 Create a New Repository

  1. Click the + icon (top-right) New repository

  2. Repository name: write brand-dhad (or anything you want)

  3. Set visibility to Public

  4. DO NOT check "Initialize with README"

  5. Click Create repository

STEP 3 Upload Your Website

  1. On the new repo page, click "Uploading an existing file"
    (or click "Add file" "Upload files")

  2. Drag and drop the index.html file into the box

  3. Scroll down Write a commit message like first upload

  4. Click Commit changes

STEP 4 Enable GitHub Pages

  1. Go to the top of your repo

  2. Click Settings

  3. On the left sidebar, click Pages

  4. Under "Source", choose Deploy from a branch

  5. Then under "Branch", choose main and click Save

Now wait 3060 seconds

STEP 5 View Your Website

  • Scroll back to GitHub Pages section in settings

  • Youll see a link like:
    https://yourusername.github.io/brand-dhad

  • Click it your website is now live
